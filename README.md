# Boot Start Project

## 목적
* 스프링 관련 실습을 하기 위한 프로젝트의 기본 구성을 갖춘다.
* 기술 별로 브랜치를 나누어 관리한다.

## 의존성
* spring web
* spring jpa + data-jpa + queryDsl
* H2
* lombok
* joda time
